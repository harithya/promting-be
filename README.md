# 🎨 AI Content Generator

Platform generator konten visual berbasis AI yang powerful dan mudah digunakan. Buat konten profesional untuk berbagai kebutuhan hanya dengan beberapa klik.

## ✨ Fitur Utama

### 🎯 Banyak Pilihan Kategori Konten

Aplikasi ini menyediakan 8 kategori konten yang lengkap dengan contoh penggunaan:

- **Promosi / Marketing** 🎯 - Poster produk, banner diskon, iklan social media
- **Edukasi / Informasi** 📚 - Infografis, tips & trik, diagram, tutorial visual
- **Event / Pengumuman** 📢 - Poster event, undangan digital, pengumuman penting
- **Hiburan / Fun** 🎉 - Meme, komik, ilustrasi lucu, konten viral
- **Motivasi / Inspirasi** ✨ - Quotes motivasi, storytelling visual
- **Lifestyle & Kuliner** 🍽️ - Review resto, travel guide, OOTD, daily vlog visual
- **Berita & Update** 📰 - Berita singkat, update produk, press release visual
- **Company / Corporate** 🏢 - Company profile, nilai perusahaan, internal campaign

### 🎭 Tone & Style yang Beragam

Setiap kategori dilengkapi dengan pilihan tone yang bisa disesuaikan:

- Persuasif & Mendesak
- Ramah & Mudah Didekati
- Profesional & Terpercaya
- Antusias & Energik
- Edukatif & Ramah
- Formal & Akademis
- Dan masih banyak lagi...

### 🤖 Support Berbagai AI Provider

- ChatGPT (OpenAI)
- Google Gemini
- Dan AI provider lainnya

### ⚙️ Customizable Output

- Sesuaikan struktur konten sesuai kebutuhan
- Atur fokus konten yang diinginkan
- Pilih visual style yang tepat
- Output yang fleksibel dan dapat disesuaikan

### 📝 Prompt Terstruktur & Mudah Digunakan

- Interface yang intuitif dan user-friendly
- Prompt yang jelas dan terorganisir dengan baik
- Panduan lengkap untuk setiap kategori
- Contoh penggunaan yang membantu

## 🚀 Cara Menggunakan

1. Pilih kategori konten yang sesuai kebutuhan
2. Tentukan tone & style yang diinginkan
3. Masukkan detail konten Anda
4. AI akan generate konten sesuai spesifikasi
5. Customize output sesuai keinginan

## 🛠️ Teknologi

- Laravel 11
- Filament Admin Panel
- JWT Authentication
- SQLite Database
- Tripay Payment Gateway Integration

## 📦 Instalasi

```bash
# Clone repository
git clone [repository-url]

# Install dependencies
composer install
npm install

# Setup environment
cp .env.example .env
php artisan key:generate

# Run migrations & seeders
php artisan migrate --seed

# Build assets
npm run build

# Start server
php artisan serve
```
